Cotação de Produtos?





O que é?

Plataforma online para importar e atualizar dados sobre produtos e promoções dos Hipermercados Atacadistas conveniados. As importações dos dados são executadas a partir de arquivos XMLs gerados e atualizados diariamente pelos Atacadistas.

Os dados importados podem ser consumidos através de uma API RESTful que disponibiliza recursos para retornar dados de localização de estabelecimentos, produtos e cotação em diversos Hipermercados Atacadistas.



Pesquisa

A API disponibiliza diversos recursos de pesquisa que podem ser consumidos por qualquer linguagem de programação, todos os dados são retornados no formato JSON:

  • Redes Atacadistas:- Pesquisar todos os estabelecimentos atadistas conveniados com a plataforma. Também é possível pesquisar estabelecimentos com dados de geolocalização, passando latitude e longitude.
  • Produtos:- Pesquisar todos os produtos de uma determinada rede atacadista a partir do nome da rede passado como parâmetro na requisição.
  • Promoções:- Pesquisar todos os produtos em promoção de uma determinada rede atacadista a partir do nome passado como parâmetro na requisição.


Cotação

É possível enviar como parâmetro uma string contendo o nome do produto desejado para cotação e receber como retorno todos os produtos da base de dados que tem relação com o parâmetro informado.

A coleção de dados retornada pela API após a cotação contém a descrição do produto, valor e a rede Atacadista que possui o produto.
